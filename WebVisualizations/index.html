Summary: Latitude vs. X

Purpose: Analyze how weather changes with respect to changes in Latitude.

Methods: 618 pairs of randomly determined unique Latitude / Longitude coordinates were generated and mapped to their nearest city using a public python program: citypy. These 618 cities were then used to return four weather data metrics from the OpenWeatherApp API: Temperature, Humidity, Cloudiness and Wind Speed. These four weather related measurements were then scatter plotted against Latitude to visualize any correlations. The results are presented on this website.
